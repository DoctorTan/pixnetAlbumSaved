# pixnetAlbumSaved
用来一键自动保存痞客邦相册,支持批量相册下载或单个相册下载
我有提供2个版本,脚本版和软件版
main分支是手动版, 需要手动打开js文件输入链接地址,怎么获取链接地址?具体看图;
1.找到一个相册或一个相册列表
